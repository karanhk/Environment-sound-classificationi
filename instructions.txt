This repository classifies audio recording into 50 categories of environmental sound.

The implementation.ipynb file is the implementation of model in python using tensorflow and librosa.
Run this file to implement the model again, all lines in this file are self explainatory.

The custom.ipynb file is to load the model and run it on custom audio file. You can load your trained model also and run on your custom audio file. In this file also
all lines are self explainatory.

The model.h5 is pretrained model which you can directly use.

The categories.csv is the csv file containing the categories, the index of category is the target (y_train) of model.
